# Algorithms Repository

Kho lưu trữ cá nhân để ghi chú và quản lý:

- Thuật toán
- Code mẫu
- Boilerplate

## Cấu trúc thư mục

- `algorithms/`: Lưu các bài toán, cách giải và ghi chú.
- `code-samples/`: Lưu các đoạn code tái sử dụng theo ngôn ngữ/chủ đề.
- `boilerplates/`: Lưu template khởi tạo nhanh cho các loại dự án.

## Quy ước gợi ý

- Mỗi thư mục con nên có `README.md` mô tả nội dung.
- Đặt tên thư mục/file rõ ràng theo ngôn ngữ hoặc chủ đề.
- Ưu tiên tách nhỏ theo domain để dễ tìm lại sau này.
